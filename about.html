Short History of HTML

HTML was first introduced in 1991 by Tim Berners-Lee.

  • 1991: HTML debut
  • 1995: HTML 2.0 standard
  • 1999: HTML 4.01
  • 2014: HTML5 finalized

Major Milestones

  1. Creation of the World Wide Web
  2. Introduction of CSS and JavaScript
  3. Adoption of HTML5

Key Terms

HTML
HyperText Markup Language, the standard for web documents.
Element
A component of an HTML document, such as <header> or <nav>.
Semantic
Using tags that describe their meaning and role in the document.
